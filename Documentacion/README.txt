# Automotora2
Proyecto Final


Autor: Carlos Da Silva
Version: 0
Fecha de inicio: 17/01/2022
Fecha de fin: 10/04/2022

Este proyecto va constatar en crear un sistema de gestion para renta de autos online (WEB).
Este proyecto cuenta con un sitema de gestion(backend) y un sitio web.
El sitio y el backendI van a estar construido principalmente con PHP y Mysql para todo lo que es el funcionamiento.
Como parte interacion del usuario vamos a usar HTML5, CSS, y JavaScript con el framework Materialize (https://materializecss.com/) 
El sistema corre bien con PHP 7.4.19, MySQL 5.7.33, Apache 2.4.47

Es compatible con php 7.0 en adelante. 
Es compatible con MySQL 5.0 en adelante.
Es compatible con MariaDB 10
